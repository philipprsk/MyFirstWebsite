# LEGO Local Landmarks Website

A modern and clean website showcasing LEGO recreations of iconic local landmarks and sights.

## Features

- **Modern Design**: Clean, responsive interface built with modern web standards
- **Interactive Gallery**: Filterable gallery system showcasing different categories of builds
- **Mobile-First**: Fully responsive design that works beautifully on all devices
- **Smooth Animations**: Engaging micro-interactions and scroll-based animations
- **Contact System**: Interactive contact form for visitor engagement

## Structure

- `index.html` - Main website structure with semantic HTML5
- `styles.css` - Modern CSS with responsive design and animations
- `script.js` - Interactive features and functionality

## Local Development

Simply open `index.html` in your web browser to view the website locally.

## Build Categories

The website showcases LEGO builds in three main categories:
- **Historic Sites**: Town Hall, Churches, Heritage Buildings
- **Modern Buildings**: Libraries, Contemporary Architecture
- **Landmarks**: Bridges, Clock Towers, Memorial Sites
